Running CUDA Samples Based on GPGPU-Sim

CUDA Samples

CUDA Samples是示范CUDA编程概念的软件包。自CUDA 4.1开始，CUDA Toolkit搭载了CUDA Samples安装包。CUDA Samples的目录结构如下：

  • 0_Simple
  • 1_Utilities
  • 2_Graphics
  • 3_Imaging
  • 4_Finance
  • 5_Simulations
  • 6_Advanced
  • 7_CUDALibraries
  • EULA.txt
  • Makefile
  • bin
  • common
